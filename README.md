# Deals and Tech News HTML Code Generator

This tool allows users to generate HTML code for deals and tech news. Users input data for deals and tech news along with associated links and files. Upon submission, HTML code with CSS styling is generated, which users can save as `index.html` to view the resulting page.

## Features

- Generate HTML code for deals and tech news.
- Include links and files in the generated HTML code.
- Easy-to-use interface with no limit on the number of generated items.
